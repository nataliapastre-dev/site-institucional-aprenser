## 📋 Sobre o projeto

O **Site Institucional Aprenser** é uma página fictícia criada para fins de estudo e prática de desenvolvimento Front-End.  
O projeto simula um site de uma escola de tecnologia, apresentando:

- Página inicial com banner hero  
- Seção de cursos  
- Seção sobre a escola  
- Mapa integrado  
- Área de contato  
- Layout moderno e responsivo  

O foco principal é o treino de estruturação, responsividade e criação de layout com aparência profissional.

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

- Estruturação semântica com **HTML5**  
- Estilização com **CSS3**  
- Organização modular usando containers e seções  
- Criação de layouts modernos  
- Responsividade mobile-first  
- Integração de **Google Maps (iframe)**  
- Uso correto de imagens, tipografia e espaçamentos

---

## 🚀 Tecnologias utilizadas

- **HTML5**  
- **CSS3**  
- **Google Fonts (Inter)**  
- **Normalize.css**  
- **Google Maps (embed)**

## 📚 Seções do site

### 🟡 **Hero Section**
Com frase motivacional, botão de ação e imagem ilustrativa.

### 🟡 **Cursos**
Exibe três cursos principais com cards responsivos, textos aprimorados e imagens ilustrativas.

### 🟡 **Nossa Escola**
Conta um pouco mais sobre a instituição, seus valores e sua proposta de ensino.

### 🟡 **Contatos**
Inclui endereço fictício, telefone, e-mail e mapa integrado via Google Maps.
